About this tool

  • Reads directly from the .docx XML — preserving italics, Small Caps, and bold text that plain-text export loses.
  • Splits each footnote into individual citation sources using legal citation-signal detection (See, Id., semicolons, etc.).
  • Returns either a formatted PDF table or Excel spreadsheet ready for cite-checking.
  • Commentary-only footnotes (no detectable citation) are included at the end with a note.
